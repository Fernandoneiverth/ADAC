Olá,

Para iniciar, entre no index.html, vá em [Adicionar Tarefa] e preencha todos os campos; após preencher vá em [Ver Tarefas] para consultar sua tarefa usando a seleção que quiser.
Caso a tarefa ainda não tenha sido feita, clique no botão [Não] para que a tarefa seja marcada como feita, e caso deseje excluí-la, clique no [Excluir].

Obrigado!
